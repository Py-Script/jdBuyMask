# GojdBuyMask V1.0 
## 功能
- [x] 检查登录
- [x] 确认是否有货
- [x] 有货自动下单
- [x] 邮件、微信通知

**注意会清空购物车**

理论可以添加无限个商品
 

## 温馨提示
- 在京东购物车结算页面设置发票为电子普通发票-个人设置支付方式为在线支付
- 地区id不知道如何获取的，请使用AreaTool.py获取


## 反馈问题

- 如果有红包先花掉再开脚本，不然可能需要支付密码
- 出现下单地址不是默认地址的，在线下一单，取getOrderInfo.action链接的cookie
- CMD界面卡住、关闭CMD的快速编辑模式就行了
